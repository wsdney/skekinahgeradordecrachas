📘 Shekinah Pro — Sistema de Crachás e Eventos

O sistema é dividido em duas abas principais: 1. DADOS & SALAS (Logística e Participantes) e 2. DESIGNER (Criação visual).

1. Gestão de Participantes e Salas (Aba 1)

Esta aba é o coração da organização do seu evento.

📥 Importação de Dados

Existem três formas de inserir participantes:

  • Excel/CSV: Arraste um arquivo para a zona de drop ou clique nela. O sistema extrai automaticamente "Inscrição" e "Nome".
  • Colagem Manual: Cole uma lista no formato Número - Nome Completo na área de texto.
  • Adição Rápida (Individual): Dentro de cada card de sala, utilize o formulário "➕ Adição Rápida". Digite o Nº e o Nome e clique em Confirmar Adição. Isso é ideal para inscrições de última hora sem precisar regerar as salas.

🏠 Criar e Gerir Salas

No painel Gestão de Salas, escolha o tipo de agrupamento:

  • Pessoas por Sala: Você define a capacidade de cada local.
  • Quantidade de Salas: Você define em quantos grupos quer dividir o total de inscritos.
  • Defina o Prefixo (ex: Quarto, Sala, Micro-ônibus).
  • Clique em Gerar Salas.
  • Organização Visual: Altere a cor de cada sala clicando nos quadradinhos coloridos no topo do card.
  • Preencha o campo Líderes para cada grupo.
  • Drag & Drop: Arraste participantes entre as salas para ajustes finos.

2. Criação do Design dos Crachás (Aba 2)

Aqui você define a identidade visual do evento.

📏 Configuração do Formato

  • Defina Largura e Altura em centímetros (o padrão comum é 10x7cm ou 9x13cm).
  • Ajuste a Borda/Moldura: Esta linha assumirá a cor da sala (ou a cor única escolhida).
  • Cor de Fundo: Use o seletor para definir a cor base do cartão.

➕ Inserção de Elementos

  • Texto: Adicione e use as tags {{NOME}}, {{INSCRICAO}} ou {{SALA}} para que o sistema preencha os dados automaticamente.
  • Imagem: Envie logos do evento ou patrocinadores.
  • QR Code Auto: Adiciona um código único baseado no número de inscrição.

⚙️ Propriedades e Fontes

Ao selecionar um texto, você pode:

  • Escolher entre diversas famílias, incluindo Arial, Bebas Neue e fontes cursivas.
  • Sublinhado: Ative a opção para uma linha elegante com distanciamento visual agradável.
  • Alinhamento: Esquerda, Centro, Direita ou Justificado.

3. Emissão de Crachás sem Salas (Fluxo Rápido)

Se o seu evento não exige divisão por quartos ou salas:

  • Importe a lista de nomes.
  • No painel ⚡ Fluxo Rápido, escolha Cor Única (o padrão é Preto #000000) ou Cores Variadas.
  • Clique em GERAR CRACHÁS DIRETO.
  • Vá para a aba Designer, monte o layout e o PDF será gerado com todos os nomes na cor escolhida.

4. Listas e Relatórios (Logística)

O sistema gera documentos automáticos baseados nos seus dados:

  • Lista de Dormitórios (PDF): Gera um documento em modo paisagem, organizado por sala, exibindo o nome da sala, a cor, o total de pessoas e o nome dos Líderes.
  • Lista Numérica: PDF com todos os participantes em ordem crescente de inscrição.
  • Lista Alfabética: PDF com todos os nomes em ordem de A-Z, incluindo a identificação da cor/sala de cada um.

5. Salvamento e Finalização

  • Auto-Save: O sistema salva cada alteração automaticamente no seu navegador.
  • Exportar/Importar JSON: Sempre baixe o arquivo .json ao terminar um design. Isso permite que você abra o projeto exatamente como parou em outro computador.
  • Gerar PDF Final: No painel direito, defina a margem da folha (padrão otimizado em 0.2cm) e o espaço entre crachás. Clique em GERAR PDF FINAL. O PDF sairá sequencial por salas, facilitando a separação física após a impressão.

🗑️ Dicas de Manutenção

  • Limpar Dados (Manter Design): Remove os nomes para um novo evento, mas guarda seu layout, logos e fontes.
  • Novo Projeto (Limpar Tudo): Reseta o sistema completamente para um início do zero.
